# Benutzerhandbuch - Neue Features

## Favoriten System

### Restaurants als Favoriten markieren

1. Navigieren Sie zu **"Restaurants"** in der Sidebar
2. Öffnen Sie ein Restaurant (click auf Details)
3. Klicken Sie auf das **Herz-Icon** neben dem Restaurant-Namen
4. Das Icon wird gefüllt (★) - Restaurant ist jetzt ein Favorit

### Gerichte als Favoriten markieren

1. Im Restaurant expandiert, finden Sie die Gerichte
2. Neben jedem Gericht gibt es ein **Herz-Icon**
3. Klicken Sie darauf, um das Gericht favorisieren
4. Das gefüllte Stern-Icon zeigt den Favoriten-Status

### Favoriten anschauen

1. Navigieren Sie zu **"Favoriten"** in der Sidebar
2. Sehen Sie alle favorisierten Restaurants
3. Sehen Sie alle favorisierten Gerichte mit Restaurant-Referenz
4. Klicken Sie auf das Herz-Icon, um Favoriten zu entfernen

## Markenverleih System

### Neue Person hinzufügen

1. Navigieren Sie zu **"Markenleihen"** in der Sidebar
2. Klicken Sie auf den Button **"Person hinzufügen"** (oben rechts)
3. Geben Sie den **Namen** der Person ein (erforderlich)
4. Geben Sie optional die **Anzahl Marken** ein, die Sie verleihen möchten
5. Klicken Sie **"Hinzufügen"**
6. Die Person erscheint mit Status "Ausstehend" (Amber-Box)

### Verleihung bestätigen/ablehnen

1. Oben im Dashboard sehen Sie "X ausstehende Anfrage(n)"
2. Für jede ausstehende Anfrage gibt es zwei Buttons:
   - **✓ (Grün)**: Verleihung akzeptieren
   - **✗ (Rot)**: Verleihung ablehnen
3. Nach Aktion wird die Person in die Hauptliste verschoben
4. Status wird entsprechend aktualisiert

### Marken-Balance ändern

1. Finden Sie die Person in der Liste
2. Klicken Sie **+** um Marken hinzuzufügen
3. Klicken Sie **-** um Marken zu reduzieren
4. Badge in der Mitte zeigt die aktuelle Anzahl
5. Änderungen werden sofort gespeichert

### Person löschen

1. Klicken Sie auf das **drei Punkte Menü** neben der Person
2. Wählen Sie **"Löschen"**
3. Die Person wird sofort entfernt
4. Statistiken aktualisieren sich

## Statistiken verstehen

### Im Markenverleih-View

- **Verliehen**: Marken, die Sie an andere verliehen haben (Summe aller positiven Balances)
- **Schulden**: Marken, die Sie schulden (Summe aller negativen Balances)
- **Gesamt**: Netto-Position (Verliehen - Schulden)
  - Grün: Sie haben mehr verliehen
  - Rot: Sie schulden mehr

### Im Dashboard-Panel

- **Offene Marken gesamt**: Summe aller Balances mit allen Personen

## Tipps & Tricks

### Schnelle Navigation

- Favoriten sind schnell erreichbar über "Favoriten" in der Sidebar
- Markenleihen wird auf der Startseite als Widget angezeigt
- Click auf Restaurant-Name öffnet Details

### Best Practices

1. **Favoriten nutzen**: Markieren Sie häufig besuchte Restaurants
2. **Regelmäßig aktualisieren**: Halten Sie Verleihungen aktuell
3. **Bestätigen Sie ausstehende Anfragen**: Um korrekte Statistiken zu haben
4. **Negative Balances**: Ein negativer Balance bedeutet, Sie schulden dieser Person

### Fehlerbehandlung

- Wenn eine Aktion fehlschlägt, erscheint eine Fehler-Nachricht
- Versuchen Sie die Aktion erneut
- Wenn das Problem persisiert, aktualisieren Sie die Seite

## FAQ

**F: Kann ich Favoriten mit anderen teilen?**
A: Aktuell nicht - Favoriten sind persönlich für jeden Benutzer

**F: Was passiert, wenn ich eine Person mit Schulden lösche?**
A: Die Person wird gelöscht, aber Sie sollten zuerst die Schulden abgleichen

**F: Wie lange dauert es, bis Änderungen gespeichert sind?**
A: Änderungen werden sofort persistent gespeichert - kein manuelles Speichern nötig

**F: Kann ich mehrere Verleihungen mit der gleichen Person haben?**
A: Nur eine aktive Verleihung pro Person - Änderungen aktualisieren den Eintrag

**F: Was bedeutet "Ausstehend"?**
A: Die Verleihung wurde erstellt, aber nicht bestätigt. Sie können sie akzeptieren oder ablehnen
